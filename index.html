  • Freelance Developer| 2017 - 2019

    (Turning Ideas into Impactful Tools)

    During my freelance tenure with KFA Business School, I designed and built a custom Sales Tracking System for Himalayan Bank and Shangri-La Development Bank. This PHP/MySQL platform automated manual data entry and provided real-time sales analytics, boosting team efficiency. Security was a top priority: I implemented role-based access controls and MD5 encryption to ensure compliance with strict data protection standards, earning praise from stakeholders for balancing usability with robust security. To create a truly user-friendly experience, I interviewed users and stakeholders to identify pain points and redesigned the interface from the ground up. The new UI simplified report generation with dynamic filters (e.g., date ranges, bank branches), cutting task time. I also authored step-by-step manuals and trained users, achieving about 95% adoption rate within two months, a testament to the system’s intuitive design. This project taught me that great software isn’t just functional — it’s empathetic. By listening to users and iterating on feedback, we built a tool that felt less like a system and more like a teammate.
